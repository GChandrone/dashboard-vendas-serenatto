# Dashboard de Clientes e Vendas - Serenatto Café e Bistrô ☕

## 📌 Sobre o Projeto
Este projeto foi desenvolvido como parte de um desafio de análise e visualização de dados. O objetivo principal é transformar dados brutos de vendas e perfil de clientes em informações visuais claras e úteis, criando um **Dashboard de Vendas e Clientes** no Microsoft Excel.

O dashboard permite uma análise eficaz do desempenho de vendas do **Serenatto Café e Bistrô**, facilitando a tomada de decisões baseadas em dados e o entendimento do público consumidor.

## 📊 Estrutura dos Dados Utilizados
O projeto foi construído a partir de uma base de dados interna do bistrô, contendo as seguintes tabelas principais:

* **Clientes (Base de Vendas):** Contém o histórico de transações, incluindo data da compra, dados demográficos do consumidor (Gênero, Estado Civil, Bairro), produto adquirido, quantidade e valor total da compra.
* **Lista de Produtos:** Tabela auxiliar contendo o ID do produto, nome (ex: Café Coado, Coxinha, Capuccino, etc.) e o custo de produção/aquisição.

## 📈 Indicadores e Análises (KPIs)
O dashboard finalizado apresenta as seguintes métricas e visualizações:
* **Número Total de Clientes:** Contagem de consumidores únicos atendidos.
* **Quantidade de Produtos Vendidos:** Volume total de itens comercializados no período.
* **Ticket Médio / Média de Valor:** Valor médio gasto por cliente.
* **Faturamento por Bairro:** Análise de receita segmentada por regiões (São Bento, República, Liberdade, Carrão, Belém).
* **Distribuição por Gênero:** Perfil do público consumidor (Masculino vs. Feminino).
* **Quantidade Mensal (Sazonalidade):** Evolução das vendas ao longo dos meses do ano.

## 🛠️ Tecnologias Utilizadas
* **Microsoft Excel:** Utilizado para tratamento de dados, criação de Tabelas Dinâmicas (`Dinâmica`) e construção do painel visual (`Dashboard Clientes`).

## 🚀 Como Reproduzir e Utilizar
Para visualizar e interagir com este dashboard em sua máquina, siga os passos abaixo:

1.  **Clone o repositório** ou faça o download dos arquivos:
    ```bash
    git clone https://github.com/seu-usuario/nome-do-repositorio.git
    ```
2.  **Abra o arquivo Excel:**
    Localize e abra o arquivo `Dashboard Clientes Serenatto Café e Bistrô - FINAL.xlsx` utilizando o Microsoft Excel (recomenda-se a versão 2016 ou superior para total compatibilidade dos gráficos).
3.  **Navegação pelas Planilhas:**
    * Acesse a aba **`Dashboard Clientes`** para visualizar o painel interativo.
    * As abas **`Clientes`** e **`Lista Produtos`** contêm os dados brutos.
    * A aba **`Dinâmica`** contém as Tabelas Dinâmicas que alimentam os gráficos.
4.  **Interação:** Utilize os filtros de segmentação de dados (caso aplicável) no próprio dashboard para filtrar as visões por mês, bairro ou gênero.
